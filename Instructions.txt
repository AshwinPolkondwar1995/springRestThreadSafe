Instruction to run this application.

Step1- open it on eclipse and update project using (Maven -> update project)
step 2- change database connection parameter (like url, user, password, schema name in url ) in hibernate.cfg.xml. 
Step-3- Run the application on server (	Run As -> Run on Server)
Step-4 - open url http://localhost:8080/SpringRestApi/doIncrement  (if server is running on other port(8080) then change accordingly in url) 
Step 5-  open jmeter and open "springRestApiTest.jmx". (File -> open and choose "springRestApiTest.jmx" )
Step 6-  After loaded successfully then click on Home Page (inside jmeter project) and verify port no and protocol(http/https). For me, it's http and 8080.
Step-7 click on start button.
Step-8 for testing you can hit again this url (http://localhost:8080/SpringRestApi/doIncrement) on browser and verify the result.
